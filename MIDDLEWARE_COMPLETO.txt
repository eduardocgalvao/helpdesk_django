🔧 COMO USAR
═══════════════════════════════════════════════════════════════════════════════

1. CRIAR DIRETÓRIO DE LOGS
   $ cd helpdesk_django
   $ mkdir logs
   $ chmod 755 logs

2. EXECUTAR TESTES
   $ python manage.py test tickets.middleware.tests

3. RODAR A APLICAÇÃO
   $ python manage.py runserver

4. VERIFICAR OS LOGS
   $ tail -f logs/middleware.log
   $ tail -f logs/activity.log
   $ tail -f logs/security.log

5. CONSULTAR DOCUMENTAÇÃO
   $ cat tickets/middleware/INDEX.md
   $ cat tickets/middleware/README.md
